这里是c.validate的使用，改插件提供input验证的底层方法

首先需要引入validate插件

var validate = util.validate;

validate中提供表单验证功能格式如下

validators = {
    isEmail: function(text){                          //验证邮箱
        var reg = /^(?:\w+\.?)*\w+@(?:\w+\.?)*\w+$/;   //正则表达式
        return reg.test(text);
    },            
    isDate: function(text){},             //验证日期
    isIdCard: function(idcard){}，        //验证身份证号码
}

验证邮箱

获取表单value后使用如下方法验证

validators['isEmail'](value)

这里是c.widget.calendar的使用，该插件提供日历选择功能

首先需要创建calendar插件

var Calendar = WidgetFactory.create('Calendar');

然后声明Calendar的一个实例

var calendar = new Calendar({
    date: {},
    title: '',
    callback: function (){},
    Months: 6,
    onShow: function (){},
    onHide:function (){}
})

注：callback为设置日期后的回调函数，Months是日历插件显示的月数，onShow和onHide由状态trigger引发

calender主要提供以下几种方法

calendar.setDate()    //设定日期
calendar.addDate()    //增加日期
calendar.removeDate() //删除日期
calendar.getDate()    //获取日期

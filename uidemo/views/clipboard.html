这里是Guider中粘贴板的操作方法

基本参数说明：

Guider.copyToClipboard
// @param content {String}         复制内容 

Guider.readFromClipboard
// @param callback {function}      读取粘贴板内容的回调

demo1：


        Guider.copyToClipboard({content: $('#input1').val()});

demo2：


        Guider.readFromClipboard({callback: function (key) {
        $('#input2').value(key);
        }});

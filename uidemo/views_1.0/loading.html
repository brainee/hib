这里是c.ui.loading的使用，该插件作为数据加载的过渡层

主要属性如下： 

// @param html {String}         自定义class

demo：loading的简单初始化

define(['app/ui/c.ui.loading'], function(Loading){
  var loading = new Loading();

  // ---------------------------
  // @description 显示Loading框
  loading.show();

  // ---------------------------
  // 业务逻辑 -- 加载数据

  // ---------------------------
  // @description 收起Loading框
  loading.hide();
});
